# DigiTools Platform

This project is a simple web platform built using React and Vite. The idea behind this project is to bring multiple useful digital tools into one place so users don’t have to switch between different websites.

## About the Project

While working on this, the focus was mainly on creating a clean UI and making the app fast and responsive. Vite helps with fast loading, and React makes the UI easier to manage.

## What it does

* Provides a collection of basic digital tools
* Easy to use interface
* Works smoothly on different screen sizes
* Fast loading and performance

## Tech Used

* React
* Vite
* JavaScript
* HTML & CSS

## How to run the project

Clone the repository:

```bash
git clone https://github.com/majumaderhrithik/-majumaderhrithik-B13-A06-digitools-platform.git
```

Go into the project folder:

```bash
cd -majumaderhrithik-B13-A06-digitools-platform
```

Install dependencies:

```bash
npm install
```

Start the project:

```bash
npm run dev
```

## Build

To create a production build:

```bash
npm run build
```
